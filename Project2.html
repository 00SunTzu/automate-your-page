The Basics of the Web and HTML

The World Wide Web

The World Wide Web is a collection of webpages, images and interactive content (Web applications) connected together by links (Hyperlinks - commonly abbreviated to link) .

Key Components of the Web:

  • The Browser on My Computer
  • The Webserver - located 'out there' - on the Cloud

These two elements communicate via HTTP (Hyper Text Transfer Protocol). The browser sends a request and the webserver returns the requested webpage.

How does the webserver know which page to the browser?

The browser requests a page by clicking on a Hyperlink which contains a reference to a URL (Uniorm Resource Locator) which is unique to each page on the web.

Webpage Overview

Webpages are documents of type HTML (HyperText Markup Language).

HTML comprises four key types of things:

  1. text (what you see)
  2. markup (what it looks like)
  3. references to other documents (video and images))
  4. links to other pages
HTML Fundamentals

The content of an HTML Document is made up of Elements.

All Elements are rectangluar in shape.

HTML <tags> are used to markup these Elements.

These <tags> tell the browser how to handle the individual elements of the webpage.

Most Elements are written with an opening tag, content and a closing tag.
eg. <tag>content</tag>.
In addition, some elements (like images and links) can have (an) attribute(s) which have names and values.
eg. <tag attributeName="value">content</tag>
There are also a number of tags which are referred to as void tags. These tags don't have either content or a closing tag.
Examples:
The image element: <img src="url" alt="text" >
(note: the 'alt' attribute is not compulsory but is important as it is what will be displayed if there is an error/break in the link to the image url. It is also helpful for blind users.)
The Line Break element: <br>

There are two major types or elements:

Inline Elements
Simple in nature and have only content and occupy only the space they need.
Block Elements
Contained within a virtual box and have height and width attributes as well as other potential.
By default they occupy the full width of the webpage.

Creating a Structured Document with HTML

Key Components of a Webpage

A webpage can be likened to a house.

HTML
representing the Structure and Content
CSS
representing the decorative Style
Javascript
representing the interactive elements within, eg. Garage Remote (beyond the scope of this course)

Each of the items above are Languages, they have their own Syntax and Rules.

There are rules to how websites behave.

  • HTML contains the Structure and Content of the webpage
  • HTML uses <tags> to create elements.
  • HTML elements are rectangular in shape (can be considered to be boxes)
  • HTML has a tree-like structure. HTML element content can be simple text, images or other elements (boxes within boxes). Similar to a family tree elements within the tree can have child elements and/or parent elements.
  • CSS (Cascading Style Sheets) contains the style of the webpage

How a webpage is displayed by the browser is controlled by the DOM (Document Object Model).

DOM - a cross-platform and language-independent convention for representing and interacting with objects in HTML.

The DOM uses the HTML elements to build the tree-like structure of the page.

Designing a Webpage - Boxify!

When given a mock-up of a webpage, start looking for boxes.

HTML elements are boxes so ..... find the elements!

  • Obvious first element - the whole page = <body>
  • Keep dividing up the page into boxes (within boxes) until everything on the page is reduced down to the most basic element types - eg. paragraphs or images

Cascading Style Sheets

What is CSS and how does it work?

CSS is code written to control the 'style' of HTML elements.

Styles are applied to Element types. The element types are all the predefined elements of HTML + any user-defined types known as classes.

In order for CSS to know which styles to apply to which elements we need to use a selector. For example: h1 or div
followed by the attributes and values we want applied to those elements contained within curly braces.

h1 {
    background-color : black;
    color: white:
}

How is CSS included on a web page?

There are 3 methods for applying styles with CSS.

  1. In-line in your HTML
    Should only be considered if there is some style attribute that needs to be applied which is exclusive to that particlar instance of that particular class/element. Otherwise it is an extremely inefficient method of coding style information.
    Within the body of your HTML <tag style="background-color: green">content<tag>
  2. Include the main block of your CSS within the <head> element of your HTML.
    Can be considered when the website in question has a very small number of HTML files.
    Add a <style> element within the <head> element. The content part of the <style>content</style> element should be entered as you would if you were creating a separate CSS file.
  3. Link your HTML to a separate CSS file within the <head> element of your HTML
    Write a separate CSS file and link it within the <head> element of your HTML. This is the preferred method when dealing with larger projects as it enables you to put all the standardized style features in one place and apply them across multiple pages giving a uniform look and feel to a whole website. From an efficiency point of view too, the whole look and feel of a website can be updated extremely quickly by making changes on the master styles contained within the website CSS file(s).
    Add a <link> element within your <head> element of your HTML on each page of your website.
    <head>
      <link rel="stylesheet" type="text/css" href="myStyleSheet.css">
    </head>
    Now in the same folder have a file called myStyleSheet.css which contains your CSS.

There are valid reasons for using each of these methods.

Code, Test, Refine

  1. Look for natural boxes
  2. Look for repeated styles & semantic elements
  3. Write your HTML
  4. Apply styles (from BIGGEST to smallest)
  5. Fix Things

Keep repeating steps 3 and 4 until the result looks like your page mock.

Variables and Strings

Variables

  • What is a variable?
    • A variable is a name we assign a value which can be used instead of that value in other Python expressions.
      • the value can be a any valid Python expression.
      • Variable values can vary.
  • Variables are used to make code more readable and to facilitate reuse.
  • What does it mean to assign a value to a variable?
    • When we assign a value to a variable we are actually 'putting the value into the variable'.
  • What is the difference between what = means in mathematics versus programming.
    What's the difference between 2 + 3 = 5 and myVariable = 5?
    • In mathematics = means the value on the left 'equals' the value on the right.
    • In programming = means 'assign' the value on the right to the variable on the left.
  • Variables can contain Strings.

Strings

  • What is a String?
    • A string is a sequences of characters surrounded by quotes
      (either single or double- it makes no difference so long as they are a pair!)
  • What is the difference between what + means in mathematics versus programming.
    What's the difference between 2 + 3 and "2" + "3"?
    • In mathematics + means the product of 2 + 3 which would return the value 5.
    • In programming + means the concatination of the strings "2" and "3" which would return the string "23".
  • Each character within a string has a position (called its index).
    • First character index = 0.
    • Last character index = -1.
  • Parts (ranges from one point to another) of a string can be accessed by using the index of the start and end position(s):
    • string[:-1] - returns the string from the start to index -1 (the end!)
    • string[0:] - returns the string from index [0] (the start!) to the end
    • string[:] - returns the string from the start to the end
    • expressions can be used in place of values for the index numbers.
    • string[1] will return an error if string has less than 2 characters!
  • It is possible to search for a sub-string within a sequence of characters using the Function .find("sub-string"). The index of the start of the first occurance of the sub-string will be output by the Function .find("sub-string").
    • By specifying an additional parameter you can search for the sub-string occurance after a particular index position.
Input -> Function -> Output

Functions

  • Functions can be used to reduce coding where something that will be done repeatedly can be reduced to a single call instead of needing to repeat multiple statments to achieve each time.
  • We pass 'Inputs' known as arguments/parameters "Into" Functions
    and get 'Outputs' known as return values "Out" of Functions.
  • Functions are made with a
    def functionName(input1, input2, ...., inputn):
    python statement(s)
    return output
    If you forget to include a return statement in your definition you will get no output from your function.
  • To use a Function you only need to code it into your python statement with the correct number and type of arguments.
    eg print functionName(input1, input2) and the return value of the Function will be output for use in the statement.
  • Variables used within a Function are independent of the rest of the program. Inputs and Output are treated as values and do not overwrite existing variables even if the same names are used. To change the variable outside of the function an assignment statement would be needed.
Decisions and Repetition: If and While

Comparing Things - Using if

Items (variables or values) of the same types can be compared. The result of any comparison test is either 'True' or 'False'. These are referred to as Boolean values.

We can use normal mathematical comparison operators < (less than), > (greater than), <= (less than or equal to), >= (greater than or equal to), != (not equal to). The addition of the ! in front of any operator will change the meaning to 'Not whatever the operator was before'. To test whether something is 'equal to' we use a double == (this helps to differentiate it from the assignment operator!)

Syntax for an if statement is thus:
if <test>:
   <block> (this block would only execute if the test evaluates to 'True')
else: (this is an optional extra if you want a second block of code to execute when the test evaluates to 'False')
   <block>

We can combine tests by using or or and operators.

Python will not evaluate a second test in an 'or test' situation if the first test evaluates to 'True'. This is a useful feature which we will use to our advantage.

Comparing Things - Using while

Syntax for a while statement is thus:
while <test>:
   <block> (this block will be repeatedly executed for as long as the test evaluates to 'True')

Use of a Break Test within our while block allows us to 'break out' of the loop if the condition evaluates to 'True'.

We can combine tests by using or or and operators.

Structured Data: Lists and For Loops

Lists

Lists are sequences of 'elements' (characters, numbers, other lists).

A string is a sequence of characters bound by quotes, a list is a sequence of elements bound by square brackets whose elements are separated by commas.

Lists can contain a mixture of element types - strings, individual characters, numbers and a list can itself be another element

Lists can be accessed in much the same way that strings are.

    Examples using string svar ='Howdy' list lvar = ['H','o','w','d','y']

  • particular elements/characters: svar[0] and lvar[0] both contain 'H'
  • ranges of elements/characters: svar[2:4] contains the string 'wd' byt lvar[2:4] contains the list ['w','d']

Mutation and Aliasing

Mutation allows us to modify an element of a list object without creating a new list object. Strings and numbers are never modified. New strings or numbers get created and assigned as needed while their old values are simply no longer accessible to us.

Aliasing is as you would expect - an alias is given to an list - another name by which the object is known. Changing the object using either name will result in the same change - IT IS THE SAME OBJECT you are changing!!

Appending and Concatinating

Lists can be added to using the apend method which adds a single new element to the end of the list.

Lists can also be concatinated using the + operator which creates a new list containing all the elements of the first list with the elements of the second list added on.

len(<list>)

len will return the number of elements in a list, the number of characters in a sting

Loops

Syntax:
for <name> in <list>:
   <block>

This code allows us to perform the same block of code to each element of a list. The element is assigned to variable name as the block commences.

<list>.index(<value>)- gives index of first occurance of value in list. Errors if no occurance found.

<value> in <list> - gives 'True' if value occurs within list.

<value> not in <list> - can be used as negative test too.
